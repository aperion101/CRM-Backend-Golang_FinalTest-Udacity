Bienvenue sur mon projet final CRM

Voici la liste et les fonctionnalités des apiRest concenrnant le projet CRM Backend

  • Créér un customer (Methods POST) : http://localhost:3000/customer

  • Obtenir la page actuelle (Methods GET) : http://localhost:3000/index

  • Obtenir tous les customers (Methods GET) : http://localhost:3000/customers

  • Obtenir un customer à partir de son id (Methods GET) : http://localhost:3000/customer/id

  • Mettre à jour un customer à partir de son id (Methods PUT) : http://localhost:3000/customer/id

  • Supprimer un customer à partir de son id (Methods DELETE) : http://localhost:3000/customer/id
